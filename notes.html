POST requests are used to make changes on the server, whereas GET request should not make changes to the server

Edit category.При нажатии на '...' категории, отправляется GET request на /categories/${cat.id}/edit Перехватывается CategoryController'ом Вызывается метод formEditCategory(). Он ловит {cat.id}, находит категорию с таким ИД и добавляет в модель. Также в модел добавляются прочие вспомогательные поля такие как action(смотри categories/form.html) и возвращает view category/form, Thymeleaf вставляет в поля формы параметры выбранной категории (т.к. мы добавили категорию в модель) При нажатии submit отправляется POST запрос на ссылку, переданную в модель в методе formEditCategory(), а именно /categories/{cat.id}
